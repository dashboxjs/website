The Builder

The Builder is a no-code presentation layer for your views and logic. It uses an intuitive left-to-right visual hierarchy for general organization and navigation.

The top bar of the Builder includes the:

  • Features navigation — Allows navigation between the different features:
    • Views — Build the application views and logic
    • Routes — A dedicated section for the routing system
    • Settings — A section for configuring the project settings
  • Project Name — Displays your configured project name. Clicking on the external link icon will open the public project.
